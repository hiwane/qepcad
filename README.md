# QEPCAD B

mirror of QEPCAD B: 
- http://www.usna.edu/Users/cs/qepcad
- https://www.usna.edu/CS/qepcadweb/B/QEPCAD.html

```
============================================================
QEPCAD B
A program for
Quantifier Elimination and Formula Simplification
in Elementary Real Algebra via
Cylindrical Algebraic Decomposition
============================================================
```

# Install

```sh
% git clone https://github.com/hiwane/qepcad.git
% cd qepcad
% make
```
- set environment variable `qe` to `${qecad}/qesource`
